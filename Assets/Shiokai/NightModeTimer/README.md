# Night Mode Timer README

## はじめに

本アセットは、VRChatのWorld製作において、いわゆる「ナイトモード」を時刻指定でOn/OffするUdonSharp製ギミックを提供するものです。
通常利用であれば、設定済みprefabをSceneに配置するだけで使用出来ます。
またコアギミックはUIに依存しない作りとなっているため、カスタマイズ次第で様々なUIで使用できます。 (ある程度以上のUI変更にはUdonSharpスクリプトの作成が必要です。)

## 動作環境

本アセットは以下の環境でパッケージングされています。

- Unity 2019.4.31f1
- VRC SDK - Base 3.1.11
- VRC SDK - Worlds 3.1.11
- UdonSharp 1.1.7

異なるUnityバージョンでの動作は保証いたしません。
またこれらより低いVRC SDK及びUdonSharpのバージョンでの正常動作は保証いたしません。
特にUdonSharp 0.2.x等では動作しませんので、ご注意ください。
